## git指令

1. 在空白文件夹底下创建git项目

   ```java
   //初始化git
   git init
   // 配置用户信息
   git config --global user.email "Your Email"
   git config --global user.name "Your Name"
   // 创建README.md文档
   echo "# document" >> README.md
   // 查看修改文件状态
   git status
   // 本地添加修改文件(add .表示将所有文件都提交，add + 文件名表示仅提交单个文件)
   git add .  或者 git add README.md
   // 本地提交修改文件
   git commit -m "修改内容"
   // 
   ```
